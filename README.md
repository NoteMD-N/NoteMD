# NoteMD

Clinical documentation platform for NHS doctors. Record consultations, get real-time transcription, and generate clinical letters automatically.

## Tech Stack

- **Frontend**: React + TypeScript + Vite + shadcn/ui
- **Backend**: Supabase (Auth, Database, Storage, Edge Functions)
- **Transcription**: Deepgram Nova-2 Medical (real-time streaming)
- **Letter Generation**: OpenAI GPT-4o
- **Hosting**: Render (static site)

## Development

```sh
npm install
npm run dev
```

## Build

```sh
npm run build
```
